# MILO (Michigan Online Liquor Ordering) Reference

This directory contains the authoritative reference for how MLCC's OLO
(Online Liquor Ordering) system works. It is the source of truth for:

- Cart validation rules (services/api/src/mlcc/milo-ordering-rules.js)
- RPA script design (services/api/src/rpa/*)
- Customer-facing cart UI expectations
- Error message phrasing
